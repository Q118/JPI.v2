The Beginner Jaspersoft Reports training provides in-depth instruction on the Ad Hoc Reports work area, specifically focusing on the file structure found in the View > Repository area, how to build Ad Hoc View reports, also known as Drag and Drop reports, as well as how to create Dashboards. This course will review how to build simple SQL reports from scratch and how to take a SQL query and turn it into a runnable Jaspersoft report. This course will also review how to modify existing reports that are found in a database.
